# 💳 Sistema de Compras com Cartão de Crédito

Este projeto foi desenvolvido como um desafio de programação em **Java** para consolidar conceitos fundamentais de Orientação a Objetos, manipulação de listas e interação com o usuário via console.

## 📝 Sobre o Desafio
O objetivo é criar uma aplicação que simule o funcionamento de um cartão de crédito. O sistema permite:
1. Definir um limite inicial para o cartão.
2. Registrar múltiplas compras (com descrição e valor).
3. Validar se o saldo é suficiente para cada compra.
4. Exibir o extrato final ordenado pelo valor das compras.

## 🚀 Tecnologias Utilizadas
* **Java 17** (ou versão superior)
* **Scanner**: Para entrada de dados do usuário.
* **List & ArrayList**: Para armazenamento das compras.
* **Collections**: Para ordenação dos dados.

## 🏗️ Estrutura do Projeto
O projeto é composto por três classes principais:
- `Compra`: Representa cada item adquirido, contendo descrição e valor.
- `CartaoDeCredito`: Gerencia o limite, o saldo disponível e a lista de compras realizadas.
- `Principal`: Classe responsável pela lógica de interação, contendo o menu e o loop de execução.
